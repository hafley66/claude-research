## Key Insights
- Bottom chart tooltips were broken because tooltip rendering ran before the weekly charts set HoverState. Moving tooltip block after all charts fixed it same-frame.
- Usage bars looked different widths because labels had variable text width pushing visual grouping. Fixed with fixed-width format and shared bar_x.
- Window resize must only trigger once on entry, not every frame, otherwise user can't resize freely.
- Linked cursor syncs crosshair between charts but egui_plot's CursorLinkGroups is private. Used Cell<Option<f64>> to carry hover x out of chart closures for navigator cursor line.
- libc::localtime_r available (libc already a dep) for local timezone timestamps without adding chrono.
- Responsive layout uses budget arithmetic: subtract each section's cost from available height, show/hide based on remaining budget.
