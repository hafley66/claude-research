## Key Insights
- Screenpipe's production pattern: focused-window-only AX walk, 200ms per-element timeout, 250ms wall-clock cutoff, OCR fallback for terminals/canvas apps
- AXorcist's performance key: container role pruning (only descend into AXGroup/AXWindow/AXWebArea etc), path-hint navigation to narrow search root before DFS
- Set AXEnhancedUserInterface=true on Chrome to force AX tree materialization
- egui Widget trait + builder pattern is the component model (struct=props, builder=optional props, ui()=render)
- No MUI equivalent exists in Rust GUI ecosystem -- fragmented across 5+ frameworks
- Rerun uses blueprint-as-data-store for retained UI state on top of immediate mode
- macOS Metal only supports PostMultiplied alpha (not PreMultiplied)
- For Chrome text extraction, a thin companion extension via Native Messaging is better than CDP or AX
