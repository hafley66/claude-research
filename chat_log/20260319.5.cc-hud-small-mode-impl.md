## Key Insights
- Haiku agent ran in claude-research worktree but edited cc-hud via absolute paths. Worktree isolation was no-op for cross-repo edits. Changes landed directly on cc-hud master.
- Haiku set forced window size every frame (960x460) which blocked fullscreen. Removed.
- egui_plot base_plot() sets label_formatter to empty string, killing tooltips. Override after base_plot or paint custom tooltips via plot_resp.transform.value_from_position().
- Chart y-scale must include cumulative max when overlaying bars + cumulative line, otherwise one series is invisible.
- fit button is useful in time mode (reset navigator zoom) and useless in turn mode (auto-fits). I had it backwards initially.
