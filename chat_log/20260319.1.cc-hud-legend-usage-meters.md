# Session: cc-hud-legend-usage-meters

## Goal
Continue building the cc-hud transparent egui overlay. This session focused on: expandable session legend with dropdown for grouped sessions, Anthropic usage % polling and charting, context meter bars, legend hover highlights on charts.

## Current State
Legend has collapsible groups (3+ sessions get dropdown, <=2 are flat). Active sessions sort to top. Context meter bars show tokens/200k as fill level with faded background. Usage chart polls /api/oauth/usage every 90s via curl, persists snapshots to ~/.cc-hud/usage_history.jsonl. Legend hover draws red VLines on time charts. Builds and runs in --big --history mode.

## Problem/Context
Previous session had flat legend with no session picker, total cost front-loaded, no usage % tracking. Needed expandable groups, de-emphasized cost, context awareness in legend bars, and plan utilization tracking.

## Solution/Approach
- Legend reworked with draw_legend_row helper. Groups with 3+ sessions get collapsible header (click to expand). Groups with <=2 sessions render flat rows. Active sessions sorted first.
- Stats line shows "X% ctx  $Y.YY  model" instead of call counts
- Color swatch bars: 8px wide, faded full-height bg + bright fill from bottom proportional to total_input/200k
- usage.rs module: reads OAuth token from macOS Keychain (security CLI), curls /api/oauth/usage endpoint, stores UsageSnapshot with 5h/7d/opus/sonnet utilization. Persists to JSONL on disk, loads history on startup, compacts every ~100 polls
- Dropped ureq dependency in favor of shelling out to curl (avoided 46 crate dep tree)
- LegendHighlight stored in egui temp data, read by chart closures to draw red VLines
- Session hover tooltip cleared when pointer enters usage/tool panel areas
- ScrollArea wrapping legend content with allocate_space for proper scroll height
- SessionData gained model field

## Key Insights
- Anthropic usage endpoint: https://api.anthropic.com/api/oauth/usage with OAuth bearer token + anthropic-beta: oauth-2025-04-20 header. Returns 5h/7d/opus/sonnet utilization percentages. Rate limited aggressively.
- OAuth token stored in macOS Keychain under "Claude Code-credentials" service, account is username. JSON with claudeAiOauth.accessToken field.
- ureq pulled in 46 crates (ring, rustls, icu_*, etc). Shelling out to curl has zero deps and works identically.
- ScrollArea with manual rect positioning requires allocate_space at end to declare content height
- egui temp storage (data_mut/get_temp) works well for cross-widget communication (legend hover -> chart VLines) within a single frame
- Session hover tooltip must be explicitly cleared when pointer enters non-chart areas, otherwise it persists from the "don't clear when stationary" fix
