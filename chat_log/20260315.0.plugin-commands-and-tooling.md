## Key Insights
- Plugin cache is a full copy, not symlink -- replaced with symlink for live edits
- set -euo pipefail + ls on empty glob = silent exit code 2, use nullglob+array instead
- LLMs can't count characters -- right-side box borders always drift, left+top+bottom fine
- /vibes tracks conversation bifurcations and drift, /map draws system topology
- Commands using only Read/Glob/Write don't prompt; bash bang commands always prompt
